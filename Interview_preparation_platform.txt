9. CarDetails.jsx page logic => """""""" 1. Getting the Car ID from URL
const { id } = useParams()
This extracts the id from the URL.
Example: /car/3 → id = 3
2. Navigation Hook
const navigate = useNavigate()
This lets you programmatically move to another page (like redirect).
Not used yet here, but useful later (e.g., "Back", "Book Now").
3. State to Store Selected Car
const [car, setCar] = useState(null)
Initially, car = null
After fetching/matching, it will store the selected car object.
4. Finding the Car (Main Logic)
useEffect(() => {
  setCar(dummyCarData.find(car => car._id == id))
}, [id])
What happens here:
Runs whenever id changes.
Searches (find) through dummyCarData.
Matches the car whose _id equals the URL id.

👉 Important:

find() returns first matching object OR undefined if not found.
== is used instead of === to allow number/string comparison.
5. Conditional Rendering
return car ? (...) : <p>Loading.....</p>
If car is found → show car details
If car is still null → show "Loading..."
🔥 Flow Summary
Page loads
Get id from URL
useEffect runs
Find matching car from dummyCarData
Save it in state
Render:
If found → show UI
Else → Loading""""""""
